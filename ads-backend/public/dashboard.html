Google AdSense & Other Providers

        

          Paste any third-party ad code (AdSense, Media.net, Ezoic, PropellerAds, etc.) directly into the
          Custom HTML Code field when creating an ad. The loader executes scripts correctly —
          including AdSense's adsbygoogle.push({}).
        


        
        

          
1Add the AdSense global script to Custom Head HTML

          
In Site Settings → Code Injection → Custom <head> HTML, paste your publisher script once:

          

            
            
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXXX" crossorigin="anonymous"></script>

          

        


        
        

          
2Create an ad — paste only the unit code in Custom HTML

          
In Manage Ads → New Ad → Custom HTML Code, paste your ad unit block. Example for a responsive unit:

          

            
            
<ins class="adsbygoogle"
     style="display:block"
     data-ad-client="ca-pub-XXXXXXXXXXXXXXXXX"
     data-ad-slot="XXXXXXXXXX"
     data-ad-format="auto"
     data-full-width-responsive="true"></ins>
<script>
     (adsbygoogle = window.adsbygoogle || []).push({});
</script>

          

        


        
        

          
Fixed-size unit (e.g. 728×90 leaderboard)

          

            
<ins class="adsbygoogle"
     style="display:inline-block;width:728px;height:90px"
     data-ad-client="ca-pub-XXXXXXXXXXXXXXXXX"
     data-ad-slot="XXXXXXXXXX"></ins>
<script>
     (adsbygoogle = window.adsbygoogle || []).push({});
</script>

          

        


        
        

          
Other providers work the same way

    •

    •
